CW¶

class Experiments.CW(**kwargs)¶

Field-swept and field-modulated continuous wave ESR experiment. The sweeps are normally performed from lower to higher fields. Despite the name, B_min may be larger than B_max, in which case the sweep direction is reversed. The data acquired on each sweep is stored separately, which allows you to separate bad sweeps (temperature fluctuations, glitches, etc.) from good ones.

After the experiment is completed, the sweep coil current is ramped to zero. This is a safety feature.

B_min¶

The lower limit of the magnetic field range [T]

B_max¶

The upper limit of the magetic field range [T]

field_steps¶

The number of points along the field axis

dwelling_time¶

The duration [s] for which the magnetic field remains stationary before a reading from the lock-in detectors is taken. If this parameter is defined as None (the default), the software will set it to \(5\tau\) where \(\tau\) is the current time constant of the lock-in detectors.

settling_time¶

The duration [s] for which each sweep is delayed after the field has been ramped to the starting field.

After acquisition, the following additional properties are defined.

data¶

The acquired data, organized in a complex-valued \(n\times m\) array, where \(n\) is the number of sweeps and \(m\) is the number of field steps.

ended¶

A textural representation of the time at which the experiment finished.

fields¶

An array of the nominal field values used during the sweep(s) [T].

lockin_phase¶

The phase setting of the lock-in detectors [degrees]

lockin_sensitivity¶

The sensitivity of the lock-in detectors [V]
