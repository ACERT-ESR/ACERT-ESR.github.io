ELDOR¶

class Experiments.ELDOR(**kwargs)¶

The Exchange Correlation Spectroscopy (2D-ELDOR/EXCSY) class implements a three-pulse exchange correlation spectroscopy / 2D electron-electron double resonance spectroscopy experiment. The three pulses are of the form: pi/2 - t1 - pi/2 - T - pi/2 - t2.

averages¶

The number of shots to be averaged for each step in the phase-cycle.

phasecycle¶

A list of phase combinations denoting the desired phasecycle. Be default, Peter’s 32 step ELDOR phase cycling is used. Full CYCLOPS phasecycle can be specified instead by setting eldor.phasecycle = PhaseCycling.FullPhaseCycle(3);

pulse_length¶

The nominal duration [s] of the pulse.

rep_rate¶

The shot repetition rate [\(s^{-1}\)]

T¶

The mixing time, or time between the center of the second and third pulse [s].

t1_inc¶

The time increment [s] used to step out the delay \(t_1\) between the center of the pulses. This increment should be positive.

t1_min¶

The minimum delay [s] between the center of the pulses.

t1_steps¶

The number of steps of the inter-pulse delay \(t_1\).

baseline_requested¶

If true, the experiment will be run twice. The second run will be a baseline or “off-field” measurement using the sweep coils to generate a fixed \(B_{0}\) field offset.

The lower limit of the magnetic field range [T]
scans¶
The number of scans (i.e experiment runs) to complete. If this number N is greater than 1, the
experiment will be re-run N times. Files will be named with a scan ID before the .hdf5 prefix
(e.g MyExperiment_03.hdf5
in_field_sweep_current¶

When baseline_requested is true, this value will be used for the “in-field” sweep current during data collection. Typically, this will be close to 0 and represents minor adjustments to \(B_{0}\).

off_field_sweep_current¶

When baseline_requested is true, this value will be used for the “off-field” sweep current when measuring the baseline. This value should be sufficiently far from 0.

After acquisition, the following additional properties are defined.

ended¶

A textual representation of the time at which the experiment finished.

field¶

The nominal magnetic field [T] at which the experiment was performed.
