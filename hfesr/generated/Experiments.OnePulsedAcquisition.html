OnePulsedAcquisition¶

Experiments.OnePulsedAcquisition(spectrometer, sequences, averages, reprate, addjmp=None)¶

This is just like PulsedAcquisition except that it sends all of the sequences and averages over all of them. It is meant to test averaging by the spectrometer.

Parameters:
  • spectrometer – The Spectrometer.Spectrometer instance
  • sequences – A sequence of pulse sequences, each (start,duration,phase)
  • averages – How many times to repeat each sequence. Can be a list.
  • reprate – How many times per second to send the sequence
  • addjmp – Whether to set the JMP channel, and the 1-based index of the pulse on which to set the JMP channel.
Returns:

The return value is an array of complex numbers.
