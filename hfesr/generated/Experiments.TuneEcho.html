TuneEcho¶

class Experiments.TuneEcho(**kwargs)¶

For tuning on signal. Phase cycles only the first pulse, but this should be enough to isolate the signal from the background.

This experiment runs until a key is pressed. It plots the following quantities:

Quantity x axis (abs, real, imag) Position Scaled up by
ringdown \(t_2\) (red, yellow, purple) -y_range/3  
signal \(t_2\) (purple) +y_range/3 self.echomag
signal norm real time (white) -y_range/1.5 self.echomag*self.running_scalefactor

Note

sphinx autodoc might show all the class attributes here as set to None. In fact, they are typically set to a value after initialization.

Note

The following should be moved to a separate rst file and re-used as appropriate

After acquisition, the following additional properties are defined.

data¶

The acquired data, organized in a complex-valued \(n\times m\) array, where \(n\) is the number of steps in the phase cycle and \(m\) is the record length.

ended¶

A textual representation of the time at which the experiment finished.

field¶

The nominal magnetic field [T] at which the experiment was performed.
